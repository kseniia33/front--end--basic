Команди:

  • git add - дозволяє додаті нові зміни у файли
  • git status - дозволяє переглянути список зміненних файлив
  • git add . - дозволяє додавати нові файли до майбутнього комміту
  • git commit - дозволяє зберигти зміни у репозиторії
  • git clone - дозволяє клонувати репозиторій
  • git push - дозволяє надіслати змиіни файлу у репозиторій
  • git branch - дозволяє додати нову гілку до існуючої
  • git checkout - дозволяє перейти до існуючої гілки
